The following Pages are made by me!

Gallery

I created the Gallery page for our website using HTML to build the web page. In this gallery there are 3 topics Nutrition & Health, Physical activity and Mental Health.

Each topic has 4 images and these images show activeties and foods that users can do to have a healthier life style.

Then, I used CSS for all the styling. For example, when the user hovers over an image, the image spins to the left a little to show that it's interactive.

Additionally, I used JavaScript to make the pictures accessible. So, when the user presses on it, the image pops up bigger and you can see a description, in this description the user can see why that certain activity or food is important.

Sitemap

In this sitemap, I have used HTML to build the webpage. I have mostly used SVG to design everything, by that i mean all the nodes and lines that connect them from the main node to the sub.

I also added links to all the main pages of the website, so when the user clicks on them, they are taken to the respective pages.

Then I used CSS for styling all the fonts, and I also added a hover effect over the nodes, so when the user hoovers over the node the node turns to a darker brown to show that it is interactive.

I have also used JavaScript for the subtrees so that when the user clicks on 'Content Pages' or 'Editors Pages,' they will pop up, and the user will be able to click on them.
